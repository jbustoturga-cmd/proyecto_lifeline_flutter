Este repositorio contiene la primera versión de Life Line, una aplicación móvil desarrollada en Flutter como parte del proyecto formativo.
La app está orientada a una tienda con productos para personas con cáncer, incluyendo un mural motivacional y un perfil de usuario.

 --Funcionalidades principales

Inicio de sesión y registro de usuario

Pantalla principal (Home) con navegación a las demás secciones

Tienda con productos cargados usando ListView.builder

Carrito de compras con productos agregados desde la tienda

Mural motivacional con imágenes cargadas desde assets

Perfil de usuario con opciones de configuración

Navegación entre pantallas usando Navigator


--Widgets utilizados (según requisitos del taller)

Row, Column, Expanded, Flexible

Navigator con rutas anónimas

TextField + TextEditingController

ListView.builder para listas dinámicas

Container con propiedades avanzadas (bordes, sombras, padding, etc.)

Assets (imágenes PNG y fuentes
